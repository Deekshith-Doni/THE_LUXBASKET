# THE LUXBASKET

Premium luxury gifting e-commerce platform built with Next.js, React, MongoDB, and Tailwind CSS.

## Features

- Product browsing with category and product detail pages
- Cart, wishlist, and checkout flows
- User authentication with NextAuth
- Admin dashboard for orders, products, and stats
- Contact, inquiry, and corporate gifting forms

## Tech Stack

- Next.js 15
- React 19
- MongoDB with Mongoose
- Tailwind CSS
- Zustand for client state

## Getting Started

Install dependencies:

```bash
npm install
```

Create a `.env.local` file with the required environment variables for authentication, MongoDB, email, and any third-party services used by the app.

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run start` - Run the production server
- `npm run lint` - Run lint checks
